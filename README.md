# ChelseaFC
Analysis of Chelsea Football Club in all games and all competitions starting with the 2019/20 season. KTBFFH.

## Team Stats Being Tracked

- Goals Scored
- Goals Allowed
- Passes Compleated
- Passes Attempted
- Shots Taken
- ShotZone
- Tackles
- Fouls
- Cards
- Saves
- Chances Created

- Goals Scored
- Goals Allowed
- Passes Compleated
- Passes Attempted
- Shots Taken
- ShotZone
- Tackles
- Fouls
- Cards
- Saves
- Chances Created

## Match Facts Tracked

- Weather
- Location
- Kick Off Time
- Record Before and After Game
- Table Position Before and After Game
